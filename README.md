# Duck Social Network

Java application that simulates a small social network using a relational database.

## Features

- User management
- Friend relationships
- Database persistence
- Data queries and filtering

## Technologies

- Java
- Object Oriented Programming (OOP)
- SQL
- Gradle

## Project Structure

- `src` – application source code
- `script-files` – SQL scripts for database
- `resource-files` – additional resources

## Purpose

The project was developed as part of a university assignment to practice backend development, object-oriented programming and database integration.
